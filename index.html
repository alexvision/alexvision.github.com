Application: Chew

Chew is an Android and iOS app that helps users find and share the best eats around them. Chew lets users find specific foods they are craving, and helps discover hot new things to eat. It is built with React Native. The code is available on github.

Application: Yodel

Yodel is a web app for managing a shared playlist of songs to auto-play at parties. It lets everyone at the party have a say on what songs they would like to see played, while letting the host control what is actually played. Yodel is built using AngularJS, Node.js and Socket.io, and the code is available here.

Application: Yep

Yep is a mobile app that enables users to plan impromptu events with their friends. It provides a simple way to get a short message out to all your friends by sending a push notification to your selected group of friends. Yep can be downloaded and installed from the source code on github.
